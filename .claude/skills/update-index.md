# スキル: 記事インデックス更新

## 呼び出し方
```
claude "記事を公開した"
claude "インデックスに追加して"
claude "articles/XXX/final.md をインデックスに追加"
```

---

## 実行手順

### Step 1: 対象記事の特定
- 直近で作業した記事フォルダを確認
- または指定された記事IDのfinal.mdを読み込む
- final.mdがなければ「draft.mdをfinal.mdにリネームする？」と確認

### Step 2: サマリー生成
final.mdを読んで以下を抽出：

```markdown
### [{記事ID}] {タイトル}
- **公開日**: {今日の日付 or ユーザーに確認}
- **URL**: {ユーザーに確認}
- **カテゴリ**: {Magazine 1-6を判定}
- **テーマ**: {一言で要約}
- **サマリー**: 
  - {記事の要点1}
  - {記事の要点2}
  - {記事の要点3}
- **キーフレーズ**: 「{記事内の印象的な一文}」
- **関連ストーリー**: {使用したSTORY-XXXを列挙}
```

### Step 3: インデックス更新
`articles/index.md` に追記：
1. 該当月のセクションに記事を追加
2. カテゴリ別クイックリファレンスにも追加
3. 統計を更新（総記事数、総文字数）

### Step 4: story-bank確認
- 記事内で使った新エピソードがstory-bank.mdにあるか確認
- なければ追記案を提示

---

## 出力例

```
articles/20260115-property-search/final.md をインデックスに追加しました。

---

### [20260115-property-search] 3ヶ月50件で成約ゼロ。都内物件探しで学んだこと
- **公開日**: 2026/01/15
- **URL**: （要入力）
- **カテゴリ**: Magazine 4: Creative Business
- **テーマ**: 都内物件探しの試行錯誤と学び
- **サマリー**: 
  - 都内民泊用物件探しで3ヶ月50件アプローチして成約0
  - いすみの時と違い「急いでない」のが問題だった
  - 不動産屋との関係構築の重要性に気づいた
- **キーフレーズ**: 「数打てば当たる、は都内では通用しなかった」
- **関連ストーリー**: STORY-001, STORY-015

---

NoteのURLを教えてもらえれば、インデックスを完成させます！

また、STORY-015（都内物件探しの試行錯誤）がstory-bankに追加されていることを確認しました。
```

---

## 自動チェック項目

インデックス追加時に以下を確認：

1. [ ] final.mdが存在する
2. [ ] タイトルが明確
3. [ ] カテゴリが適切
4. [ ] サマリーが3点以上
5. [ ] キーフレーズが印象的
6. [ ] 関連ストーリーがstory-bankに存在
7. [ ] 月別セクションに追加済み
8. [ ] カテゴリ別リファレンスに追加済み
9. [ ] 統計を更新済み
