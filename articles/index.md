# 記事インデックス（Article Index）

公開済み記事の一覧とサマリー。
Claudeは新規記事生成時にこのファイルを参照し、過去記事との重複を避け、関連記事を引用できる。

---

## 使い方

### Claude向け
- 新規記事生成時、まずこのファイルをチェック
- 関連テーマがあれば「以前〜について書いたけど」と繋げる
- 詳細が必要な時だけ `articles/{ID}/final.md` を参照

### 人間向け
- 記事公開後、このファイルに追記する
- サマリーは3-5行程度でOK

## テンプレート

記事公開後、以下の形式で追記：

```markdown
### [{記事ID}] {タイトル}
- **公開日**: YYYY/MM/DD
- **URL**: https://note.com/xxx/n/xxx
- **カテゴリ**: Magazine 1-6 のどれか
- **テーマ**: 一言で
- **サマリー**: 
  - {記事の要点1}
  - {記事の要点2}
  - {記事の要点3}
- **キーフレーズ**: 「{記事内の印象的な一文}」
- **関連ストーリー**: STORY-XXX, STORY-XXX
```

---

## 記事追加の自動化

記事公開後、Claudeに依頼：
```
claude "今公開した記事をインデックスに追加して"
```

→ Claudeが `articles/{ID}/final.md` を読んでサマリーを生成し、このファイルに追記
